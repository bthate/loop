**NAME**

``loop`` - run os.popen in sub directories.

**SYNOPSIS**

``loop [dir] <txt>``

**DESCRIPTION**

The ``loop`` prgram allows you to execute os.popen commands in subdirectories.

**INSTALL**

| ``pip3 install loop --upgrade --force-reinstall``
|
